## Prerequisites

* Linux, Macos, WSL
* PHP >= 8.0
* Make
* Git

## Setup

```bash
git clone git@github.com:Roman52/php-project-lvl1.git
cd Brain-Games

# Install
make install
```

## How to play:
#### There are 5 games:

1. Brain Calc - What is the result of the expression.

```bash
# Run the game
make brain-calc
```

#### Asciinema brain-calc
https://asciinema.org/a/iR2JmHvnO17O4Hp4Z7RGZsON7

2. Brain Even - Checking if a number is even.

```bash
# Run the game
make brain-even
```

#### Asciinema brain-even
https://asciinema.org/a/GKurqK7EV9aE8NOdtyuXaipKl

3. Brain GCD - Find the greatest common divisor of given numbers.

```bash
# Run the game
make brain-gcd
```

#### Asciinema brain-gcd
https://asciinema.org/a/VH1zBJIOcur6nBjvgwa2uqokT

4. Brain Progression - Guessing What number is missing in the progression.

```bash
# Run the game
make brain-progression
```

#### Asciinema brain-progression
https://asciinema.org/a/ApghwPwUQNkNdEn7s7KFinbqI

5. Brain Prime - Guessing if given number is prime or not.

```bash
# Run the game
make brain-prime
```

#### Asciinema brain-prime
https://asciinema.org/a/9wz0CCYXlQBslpGZhrTJzOUTL
