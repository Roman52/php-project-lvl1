## Prerequisites

* Linux, Macos, WSL
* PHP >= 8.0
* Make
* Git

## Setup

```bash
git git@github.com:Roman52/php-project-lvl1.git
cd php-project-lvl1

make install
```

## Run linter

```sh
make lint
```
